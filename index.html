1. Email

Working of E-mail

The email refers to the electronic mail means of communication of sending and receiving messages over the internet.

Email is the most common form of communication nowadays.

An email has significantly evolved over the past couple of years.

There are now stronger sync and messaging features along with stronger security and spam-related features.

Components of an E-mail

Sender: The sender creates an email in which they record the info that needs to be transferred to the receiver.

Receiver: The receiver gets the info sent by the sender via email.

Email address: Like a house address where the communication arrives for both sender and receiver.

Mailer: A program like Gmail or Outlook used to read, write, and manage emails.

Mail server: Responsible for sending, receiving, managing, and recording all the emails.

Data is processed by their respective mail programs and delivered to users.

SMTP (Simple Mail Transfer Protocol)

SMTP uses the internet network connection to send and receive email messages.

Protocols of E-mail

POP (Post Office Protocol): Drops the email at the mail provider and handles delivery. Works offline after sending.

IMAP (Internet Message Access Protocol): Supports two-way communication, stores conversations in a database, and notifies senders when emails are read.

Sending an Email

  1. Open the email client and click on "compose new email".
  2. Enter the recipient's address in the "To" field.
  3. Enter a subject in the "Subject" field.
  4. Attach any files if needed using the "Attach file" button.
  5. Type the message in the body.
  6. Click "Send".

Receiving an Email

  1. Open the email client and go to the inbox.
  2. Click on the email to open it.
  3. Read the content.
  4. Click on any attachments to open or download.

Forwarding an Email

  1. Open the email client and go to the inbox.
  2. Click on the email to forward.
  3. Click the 'Forward' button or select "Forward" from the menu.
  4. Enter the recipient's email in the 'To' field.
  5. Add a message if needed.
  6. Attachments will be included.
  7. Click "Send".

Zoom and Google Meet


Zoom and Google Meet are popular digital communication tools that allow users to conduct online meetings, webinars, and video conferences.
These platforms heavily rely on Voice over Internet Protocol (VoIP) technology, which enables voice calls using an internet connection instead of an analog phone line.
This technology facilitates real-time audio communication during meetings on Zoom and Google Meet.
VoIP ensures high-quality audio transmission for efficient communication regardless of location.


Steps to Create and Organize a Meeting in Zoom


Log in to a Zoom account.


Click on the "Schedule a Meeting" button.


Fill in the meeting details: date, time, and duration.


Enter a meeting name and an optional description.


Choose settings for video, audio, and screen sharing.


Select the appropriate time zone.


Choose whether to require a meeting password.


Click the "Schedule" button to create the meeting.


Share the meeting link or invite participants via email.


At meeting time, open Zoom, go to the "Meetings" tab, and click "Start" next to the scheduled meeting.

Google Meet

Steps to Create and Organize a Meeting in Google Meet


Log in to your Google account.


Open Google Calendar and click the "+" button to create a new event.


Enter a title and select the date and time for the event.


Click on the "Add conferencing" button.


Select the settings icon to choose video call options.


Click the "Save" button to create the event and send invites.


Share the event link or invite participants via email.

<!DOCTYPE html>
<html>
<head>
  <title>Registration Form</title>
</head>
<body>

<h1>Registration Form</h1>

<form onsubmit="return validateForm()">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required><br><br>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required><br><br>

  <label for="password">Password:</label>
  <input type="password" id="password" name="password" minlength="8" required><br><br>

  <label for="confirm-password">Confirm Password:</label>
  <input type="password" id="confirm-password" name="confirm-password" minlength="8" required><br><br>

  <label for="phone">Phone Number:</label>
  <input type="tel" id="phone" name="phone" pattern="[0-9]{10}" required><br><br>

  <input type="submit" value="Register">
</form>

<script>
function validateForm() {
  let name = document.getElementById("name").value;
  let email = document.getElementById("email").value;
  let password = document.getElementById("password").value;
  let confirmPassword = document.getElementById("confirm-password").value;
  let phone = document.getElementById("phone").value;

  if (name === "") {
    alert("Please enter your name.");
    return false;
  }

  let emailRegex = /^[a-z0-9]+@[a-z]+\.[a-z]{2,3}$/;
  if (!emailRegex.test(email)) {
    alert("Please enter a valid email address.");
    return false;
  }

  if (password === "") {
    alert("Please enter your password.");
    return false;
  }

  if (password.length < 8) {
    alert("Your password must be at least 8 characters long.");
    return false;
  }

  if (confirmPassword === "") {
    alert("Please confirm your password.");
    return false;
  }

  if (password !== confirmPassword) {
    alert("Passwords do not match.");
    return false;
  }

  let phoneRegex = /^[0-9]{10}$/;
  if (!phoneRegex.test(phone)) {
    alert("Please enter a valid 10-digit phone number.");
    return false;
  }

  alert("Registration successful!");
  return true;
}
</script>

</body>
</html>

  <form>
    Number 1: <input type="number" id="number1"><br><br>
    Number 2: <input type="number" id="number2"><br><br>
    <input type="button" value="Calculate" onclick="performOperations()">
  </form>
  <p id="addition"></p>
  <p id="subtraction"></p>
  <p id="multiplication"></p>
  <p id="division"></p>

  <script>
    function performOperations() {
      var num1 = parseFloat(document.getElementById("number1").value);
      var num2 = parseFloat(document.getElementById("number2").value);

      document.getElementById("addition").innerHTML = "Addition: " + (num1 + num2);
      document.getElementById("subtraction").innerHTML = "Subtraction: " + (num1 - num2);
      document.getElementById("multiplication").innerHTML = "Multiplication: " + (num1 * num2);
      document.getElementById("division").innerHTML = num2 !== 0 ? "Division: " + (num1 / num2) : "Division: Cannot divide by zero";
    }
  </script>
</body>
</html>
